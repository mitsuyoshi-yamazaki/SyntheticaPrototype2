## 概要

このファイルは、Synthetica仕様v3.0.0に関する未解決の課題とTODO項目を管理します。

## 記載方法

- 「一覧」の項に、TODOと課題を共に記載する（優先度順は後で調整）
- 各項目は以下の形式で記載：
  - 課題: `[課題-番号] 課題名`
  - TODO: `[TODO-番号] タスク名`
- より詳細な内容がある場合には、「詳細」の項に、対応する番号（課題-番号 または TODO-番号）で項目を作成し、記載する
